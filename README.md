## âš™ï¸ Setup and Installation

### Prerequisites
- **Node.js** and **npm** installed.
- **Java JDK** (version 17 or higher) installed.
- **MySQL** installed and running.

### Steps

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/Srinivas-Vengaldas/DC-Book-Store.git](https://github.com/Srinivas-Vengaldas/DC-Book-Store.git)
    cd DC-Book-Store
    ```

2.  **Database Setup**
    - Create a MySQL database named `bookstore_db`.
    - Update the `application.properties` file in the backend folder with your MySQL username and password.

3.  **Backend Setup**
    Navigate to the backend directory and run the application:
    ```bash
    cd backend
    ./mvnw spring-boot:run
    ```

4.  **Frontend Setup**
    Open a new terminal, navigate to the frontend directory, and install dependencies:
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
